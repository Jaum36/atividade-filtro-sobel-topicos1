## 📌 Sobre o Projeto

Este repositório contém a **Tarefa 1** da disciplina de **Tópicos 1 (2026.1)**. O objetivo central é a implementação e análise do **Filtro de Sobel**, um operador diferencial utilizado para a detecção de bordas em imagens digitais. 

O projeto demonstra como calcular as variações de intensidade luminosa nos eixos horizontal (X) e vertical (Y), combinando-as para destacar os contornos dos objetos presentes na imagem.

## 🛠️ Tecnologias e Bibliotecas

A atividade foi desenvolvida em **Python** utilizando o ambiente interativo do Jupyter Notebook, com as seguintes bibliotecas:

* **[NumPy](https://numpy.org/)**: Processamento de matrizes e cálculos matemáticos.
* **[OpenCV](https://opencv.org/)**: Leitura e manipulação de imagem.
* **[Matplotlib](https://matplotlib.org/)**: Visualização e exibição dos resultados.
* **[Pillow (PIL)](https://python-pillow.org/)**: Suporte para diferentes formatos de imagem.
