📸 Detecção de Bordas com Filtro de SobelAtividade de Tópicos Especiais I (2026.1)Implementação e análise do operador de Sobel para extração de características em imagens digitais.📖 Sobre o ProjetoEste repositório contém a implementação da Tarefa 1 da disciplina de Tópicos Especiais I. O objetivo principal é aplicar o Operador de Sobel, um algoritmo clássico de processamento de imagens utilizado para detectar bordas, destacando regiões de alta frequência espacial que correspondem a transições de intensidade.🔍 O que é o Filtro de Sobel?O operador utiliza dois kernels $3 \times 3$ que são convoluídos com a imagem original para calcular aproximações das derivadas — uma para alterações horizontais e outra para verticais:$G_x$ (Horizontal): Destaca bordas verticais.$G_y$ (Vertical): Destaca bordas horizontais.Magnitude: $G = \sqrt{G_x^2 + G_y^2}$📂 Estrutura do RepositórioTopicos1-2026.1-Tarefa1.ipynb: Notebook principal com todo o código, explicações e visualizações.cano.webp: Imagem de teste utilizada para demonstrar a eficácia do filtro em estruturas cilíndricas e metálicas.📊 Resultados EsperadosAo processar a imagem cano.webp, o algoritmo gera:A imagem em tons de cinza.O mapa de gradiente horizontal e vertical.A magnitude total do gradiente (bordas combinadas).👥 EquipeDesenvolvido por:<table align="center"><tr><td align="center"><a href="https://www.google.com/search?q=https://github.com/jaum36"><img src="https://www.google.com/search?q=https://github.com/jaum36.png" width="150px;" alt="Foto do João Victor"/><sub><b>João Victor</b></sub></a></td><td align="center"><a href="#"><img src="https://www.google.com/search?q=https://ui-avatars.com/api/%3Fname%3DYago%2BFeitoza%26size%3D150%26background%3D0D8ABC%26color%3Dfff" width="150px;" alt="Foto do Yago Feitoza"/><sub><b>Yago Feitoza</b></sub></a></td></tr></table><p align="center">Universidade Federal - Curso de Engenharia/Ciência da Computação</p>tilizada
