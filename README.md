# Atividade 1: Extração de Bordas com Filtro Sobel
## Disciplina: Tópicos para Computação 1 (2026.1)

Este repositório contém a implementação da primeira tarefa prática da disciplina de **Tópicos para Computação 1**, ministrada pela **Profa. Dra. Elloá B. Guedes** na **Escola Superior de Tecnologia (EST/UEA)**.

## 📋 Descrição do Projeto

O objetivo principal desta atividade é aplicar o **Filtro Sobel**, um extrator de bordas clássico em processamento de imagens, para ilustrar as operações matemáticas envolvidas em algoritmos tradicionais de visão computacional. O projeto aborda:

* Abertura e manipulação de imagens em tons de cinza.
* Realização manual do processo de **convolução** em ambiente computacional.
* Visualização dos resultados e construção de mapas/vetores de características resultantes da extração.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido em Python, utilizando um ambiente de Jupyter Notebook, com as seguintes bibliotecas:

* **OpenCV (`cv2`)**: Para leitura e manipulação de imagens.
* **NumPy**: Para operações matriciais e processamento numérico.
* **Matplotlib**: Para visualização dos resultados e gráficos.
* **PyTorch**: Utilizado para operações de convolução (`torch.nn.functional`).

## 📂 Estrutura do Repositório

* `Topicos1-2026.1-Tarefa1.ipynb`: Notebook contendo todo o código-fonte, descrições teóricas e visualizações da atividade.
* `cano.webp`: Imagem utilizada como exemplo para a aplicação dos filtros.
